Installation

Prerequisites

Before installing, ensure you have conda and pip installed.

Install via Conda

The easiest way to install LIPID+ is directly from PyPI. 

# Install the latest version
$ conda create -n lipid_plus python=3.10
$ conda activate lipid_plus
$ pip install lipid_plus
$ pip install -r requirements.txt

# Upgrade to the latest version
$ pip install --upgrade lipid_plus

Data Requirements

As input, we require a feature table (formatted as a .csv file) that includes the following columns: precursor_mz, ion_mode, and MS2.

If you have information on adducts or classifications, please include these as additional columns named adduct or class.

If you do not currently have a feature table, you can generate one by processing your raw LC-MS data. Tools such as MS-DIAL, MZmine, or XCMS can be used for this feature detection process.

Should your feature detection software require mzML files as input, and you only have data in a vendor format (such as .raw), you must first use the MS Convert tool (part of ProteoWizard) to convert your raw data to the mzML format.

Your .csv feature table must adhere to the following data types. Please note the special string format required for the MS2 column.

  • precursor_mz (Float)

    The m/z of the parent ion.
    Example: 154.1234

  • ion_mode (String)

    Must be exactly "Positive" or "Negative".

  • MS2 (String/JSON)

    A JSON-formatted string of [mz, intensity] lists.
    Example: "[[80.123, 1500.4], [102.456, 9500.7]]"

  • class (Optional, String)

    The compound classification.
    Example: "Lipid" or "Unknown"

  • adduct (Optional, String)

    The adduct annotation.
    Example: "[M+H]+" or "[M-H]-"
